WIFI & HOTSPOT

Dewasa ini, kebutuhan akan internet di INDONESIA terus meningkat pesat. Internet menjadi kebutuhan pokok masyarakat yang dibutuhkan kapanpun dan dimanapun, penyediaan layanan Wifi dan Hotspot adalah hal yang tepat untuk dilakukan oleh anda yang mulai menjalankan bisnis atau usaha yang membutuhkan kehadiran pengunjung. Penambahan layanan Wifi dan Hotspot tentunya akan menjadi daya tarik tersendiri bagi pengunjung untuk datang ke tempat usaha anda, tentunya kepuasan para pengunjung adalah target utama anda. Maxindo memperkenalkan layanan Wifi dan Hotspot bagi usaha anda, dengan pelayanan kami selama 24 Jam setiap hari tentunya kualitas Wifi dan Hotspot pada Usaha anda akan terus terjaga. Layanan Wifi dan Hotspot yang kami sediakan juga dapat disesuaikan dengan keinginan anda. Kami akan memberikan penawaran layanan ini dengan Harga yang Khusus bagi anda yang berada pada area JABODETABEK, BANDUNG, dan SURABAYA dan menghubungi Marketing kami sekarang juga.

CO-LOCATION SERVER

Memperkenalkan layanan co-location dari Maxindo, sebuah layanan yang dapat membantu anda untuk kepentingan penempatan server dan koneksi internet, domestic peering IIX antara perusahaan Anda dan Media Global Internet. Lokasi penempatan Rak Server yang kami tawarkan berada pada Gedung IDC (Indonesia Data Center)

  • Rack Co-location
    Rack Colocation adalah jasa penyewaan rak untuk penempatan perangkat server maupun perangkat lain milik pelanggan termasuk di dalamnya koneksi internet ke backbone Internet/IIX (Optional) dan tenaga listrik.
  • Server Co-location
    Server Colocation, merupakan subset dari jasa Rack Colocation, yaitu jasa penyewaan ruang pada rak untuk penempatan perangkat server pelanggan.
  • Dedicated Server Co-location
    Dedicated Server Colocation adalah penyedia jasa Co-location dengan platform software dari Microsoft, perangkat keras server dan koneksi ke internet/IIX menggunakan bandwidth sesuai kebutuhan. Melalui dedicated server colocation, pelanggan tidak perlu menyediakan perangkat software, membeli software, melakukan instalasi dan perawatan. Pelanggan cukup membayar biaya bulanan sesuai dengan fasilitas yang digunakan.
